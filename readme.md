The Employee Payroll System is a desktop-based application built using Java, JavaFX, and SQLite. It automates the management of employee records and salary computation. The system allows administrators to add, view, update, and delete employees, and automatically generates salary slips in PDF format.
The project follows MVC architecture, ensuring clean separation of business logic, data handling, and UI.
The application is lightweight, user-friendly, and suitable for small organizations.
	
Conclusion & Future Scope
The Employee Payroll System provides an efficient way to manage employee details, calculate salaries, and generate salary slips automatically. It reduces manual work, minimizes errors, and offers a simple and user-friendly interface.
In the future, the system can be expanded with features like multi-user login roles, attendance tracking, automated monthly payroll, cloud database support, and email-based salary slip delivery. These enhancements would make it a more complete and scalable HR management solution.
